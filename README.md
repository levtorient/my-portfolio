# Portfolio Website

A minimalist portfolio website built with Next.js, React, TypeScript, and Tailwind CSS. This single-page application showcases personal information, skills, projects, and team members with a clean, modern design.

## Features

- **Single Page Application**: Smooth scrolling navigation between sections
- **About Section**: Personal introduction and quick facts
- **Skills Section**: Frontend, Backend, and Tools showcase with technology tags
- **Projects Section**: Project portfolio with descriptions and tech stacks
- **Team Section**: Team member profiles and roles
- **Responsive Design**: Optimized for all device sizes
- **Minimalist Styling**: Clean typography and modern layout using Tailwind CSS

## Tech Stack

- **Framework**: Next.js 15.5.3
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Package Manager**: npm

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `src/app/page.tsx`. The page auto-updates as you edit the file.

## Customization

The portfolio is designed to be easily customizable:

1. **Personal Information**: Update the hero section, about section, and quick facts in `src/app/page.tsx`
2. **Skills**: Modify the skills categories and technology tags to match your expertise
3. **Projects**: Replace the placeholder projects with your actual work, including links to live demos and GitHub repositories
4. **Team Members**: Update team profiles with actual team member information and photos
5. **Styling**: Adjust colors, fonts, and layout in the Tailwind classes
